# ACTIVA 0.3.0 – třetí balík

## Skupinové a kooperativní aktivity

Přibyly moduly:

- učební stanoviště,
- skládankové učení,
- expertní skupiny,
- papírová úniková hra,
- tisknutelná desková hra,
- rolové kartičky,
- případová studie,
- debatní sada.

Výstupy obsahují samostatně pochopitelné instrukce, čas, role, řešení a společný výstup. Rozsáhlejší sady se automaticky rozdělí na více stran.

## Obraz, diagramy a mapy

Přibyly moduly:

- popis obrázku,
- doplňování diagramu,
- slepá mapa.

Učitel nahraje vlastní PNG, JPEG nebo WebP. Aplikace obrázek lokálně zmenší a uloží do projektu. Očíslované body se přidávají kliknutím přímo do náhledu a jejich přesné popisky se upravují v textovém editoru.

## Data a čas

- graf z dat podporuje sloupcové a spojnicové zobrazení,
- časová osa vytváří vystřihovací kartičky,
- žákovská i učitelská verze používají stejná zdrojová data.

## Rozsah

ACTIVA nyní obsahuje **39 typů aktivit** ve čtyřech hlavních skupinách:

1. univerzální,
2. předmětové,
3. skupinové,
4. obrazové, mapové a datové.

## Kompatibilita

- aplikační ID zůstává `activity-builder`,
- oprávnění `ACT-01` se nemění,
- projekty 0.1.0 a 0.2.0 se načítají,
- minimální doporučená verze AI Studia je 0.17.6.
