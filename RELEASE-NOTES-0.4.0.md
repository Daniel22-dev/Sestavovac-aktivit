# ACTIVA 0.4.0 · Balík 4

## Nové funkce

- projekční režim vedený učitelem,
- časovač 30 s / 1 / 2 / 5 min,
- postupné odhalení řešení,
- klávesová navigace a režim celé obrazovky,
- až čtyři týmy a náhodný výběr týmu,
- označení odpovědi správně / částečně / chybně,
- souhrn lekce a lokální anonymní historie,
- osobní knihovna materiálů,
- oblíbené, filtrování a hledání,
- čtený školní katalog schválených ukázek,
- export/import knihovny,
- sdílecí JSON a přenosný textový kód,
- samostatný interaktivní HTML soubor.

## Bezpečnost a soukromí

- projekce nevyžaduje žákovské účty ani telefony,
- statistiky neobsahují jména žáků,
- školní katalog se načítá pouze pro čtení,
- obrazové podklady se neposílají AI,
- při nedostupné IndexedDB se použije záložní lokální nebo dočasné úložiště.

## Kompatibilita

- projektové schéma zůstává `activa-project-v1`,
- kompatibilita s verzemi 0.1.0, 0.2.0 a 0.3.0,
- všech 39 dosavadních modulů, tisk a diferenciace zůstávají zachovány,
- AI Studio vyžaduje od této verze minimálně 0.17.7.
