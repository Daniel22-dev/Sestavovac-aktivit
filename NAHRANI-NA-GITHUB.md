# Nahrání ACTIVA 0.4.0

1. Otevřete stávající repozitář `Sestavovac-aktivit`.
2. Nahraďte jeho obsah soubory z balíku ACTIVA 0.4.0. Zachovejte strukturu složek.
3. Potvrďte změny a vyčkejte na zelené GitHub Actions / GitHub Pages.
4. Otevřete nasazenou ACTIVA a proveďte tvrdé obnovení stránky.
5. V manuálu ověřte verzi `0.4.0`, údaj `39 modulů aktivit` a sekce Projekce, Knihovna a Sdílení.
6. Vyzkoušejte školní katalog, projekční režim a export samostatného HTML.
7. Teprve poté nahrajte aktualizaci AI Studia GHRAB 0.17.7.

Aplikační ID `activity-builder`, repozitář a přístup `ACT-01` se nemění. Projekty z verzí 0.1.0 až 0.3.0 se načtou automaticky.

## Školní katalog

Soubor `src/school-library/library.json` je zdroj katalogu. Do něj patří pouze anonymizované, věcně zkontrolované materiály bez osobních údajů žáků. Po změně spusťte `npm test` a nasaďte novou verzi aplikace.
