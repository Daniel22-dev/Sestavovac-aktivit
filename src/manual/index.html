ACTIVA 0.5.0 · REDAKČNÍ STUDIO

Od učiva k profesionálnímu materiálu bez ruční sazby.

Tento manuál provází celou aplikací: od prvního vložení textu přes 39 typů aktivit a diferenciaci až po tisk, projekci, knihovnu, diagnostiku a budoucí připojení školního serveru.

Aktuální pro ACTIVA 0.5.0 · GHRAB QA 1.0.2
ZAČNĚTE ZDE

Rychlý start

1Vložte učivoTéma, souvislý text, seznam pojmů nebo vlastní dvojice. Nepoužívejte skutečná jména žáků.
2Vyberte formátyACTIVA nabídne vhodné aktivity. Doporučení můžete libovolně změnit.
3Vytvořte návrhPoužijte Gemini AI, nebo místní ukázkový režim bez odeslání dat.
4Zkontrolujte obsahUpravte jednotlivé otázky, odpovědi, body, úroveň a rozložení.
5Vytiskněte nebo promítněteStáhněte PDF přes systémový dialog nebo spusťte projekci bez telefonů.
6Uložte a sdílejteProjekt uložte do osobní knihovny nebo předejte kolegovi jako balíček.
PRACOVNÍ POSTUP

Vytvoření prvního materiálu

1. Zdroj a cíl

Vyplňte předmět, ročník, téma, cíl a čas. Cíl formulujte jako pozorovatelný výkon, například „žák rozliší příčiny a důsledky“.

Nejlepší výsledky vznikají z kvalitního zdrojového textu a jasného cíle. Pouhé jednoslovné téma je možné, ale AI má méně opory pro věcnou přesnost.

2. Typy aktivit

Vyberte nejvýše pět formátů pro jeden pracovní list. Při delší lekci je vhodnější vytvořit dvě kratší sady než jednu přeplněnou.

3. Generování

Režim AI vytváří obsah podle zdroje. Místní návrh slouží k ověření vzhledu a nevyžaduje API klíč. Před odesláním AI musí projít kontrola soukromí.

4. Editor

Klikněte na aktivitu v levém seznamu. V pravém inspektoru upravte zadání, položky, řešení, body a prostor pro odpověď. Každou aktivitu lze upravit samostatně bez přegenerování celé sady.

BEZPEČNOST

AI, API klíč a soukromí

  • API klíč se posílá pouze službě Google Gemini.
  • Na sdíleném počítači používejte uložení jen pro relaci.
  • Zdrojový text se neposílá do školního katalogu automaticky.
  • Obrázkové podklady se do Gemini neposílají.
  • Statistiky projekce nevyžadují jména žáků.
  • Výstup AI vždy věcně a pedagogicky zkontroluje učitel.
Kontrola soukromí je ochranná pomůcka, nikoli záruka. Před odesláním odstraňte jména, e-maily, telefonní čísla, adresy, rodná čísla a citlivé okolnosti.

MATERIÁLY

Osobní knihovna, školní katalog a sdílení

Osobní knihovna

Ukládá projekty v aktuálním prohlížeči. Používá IndexedDB a při nedostupnosti bezpečný místní záložní režim.

Školní katalog

V bezserverové verzi je katalog verzovaný soubor, který spravuje administrátor. Položky se otevírají jako vlastní kopie, takže originál nelze přepsat.

Sdílení

Materiál lze přenést jako JSON balíček, textový kód ACTIVA4 nebo samostatnou interaktivní HTML verzi bez API klíče.

Před sdílením zkontrolujte název, poznámky, obrázky i volbu „uložit původní zdrojový text“.
BEZSERVEROVÝ PROVOZ

Zálohy, migrace a obnova

  • Pravidelně exportujte osobní knihovnu.
  • Důležité projekty ukládejte také jako samostatné JSON soubory.
  • Projekty 0.1–0.4 se při načtení normalizují do aktuálního modelu.
  • Vymazání dat prohlížeče odstraní místní knihovnu.
  • API klíč není součástí exportu projektu ani diagnostiky.
  • Při naplnění úložiště zmenšete obrázky nebo rozdělte knihovnu.
PŘIPRAVENO NA DALŠÍ ETAPU

Budoucí školní server

ACTIVA 0.5.0 zůstává bezserverová, ale používá oddělené rozhraní activa-persistence-adapter-v1. Budoucí konektor může nahradit lokální ukládání a přidat přihlášení, sdílený katalog, synchronizaci a centrální zálohy bez přestavby editoru.

Serverový režim je v této verzi záměrně vypnutý. Aplikace se nesnaží připojovat k žádnému neexistujícímu školnímu backendu.

KVALITA

Interní testy a diagnostika

Stránka Interní testy kontroluje běžící aplikaci, 39 modulů, ukládání, tisk, manifest, PWA, manuál, serverovou připravenost a napojení na AI Studio. Testy neodesílají data.

Pro vydání se navíc používá GHRAB QA 1.0.2: technická, bezpečnostní, PWA, kritická, vizuální a kombinatorická brána. Výsledek READY vyžaduje také ruční kontrolu screenshotů a smoke test skutečně nasazené verze.

Technický protokol v aplikaci neobsahuje zdrojový text ani API klíč. Správce jej může zkopírovat nebo stáhnout jako JSON.
